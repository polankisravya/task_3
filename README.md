# Task 3: Networking Basics for DevOps

## Objective
Understand basic networking concepts and how systems communicate over a network using Linux CLI tools.

## Tools Used
- Linux Command Line Interface

## Steps Performed
1. Checked system IP address using `ip a`
2. Tested network connectivity using `ping`
3. Inspected open ports using `ss`
4. Verified DNS resolution using `nslookup`
5. Traced network path using `traceroute`
6. Simulated network failure by disabling and enabling interfaces
